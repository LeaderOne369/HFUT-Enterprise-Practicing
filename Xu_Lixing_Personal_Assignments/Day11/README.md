# 第十一次作业

- 1、实现一个简易的音频/视频播放器，支持常见的音频文件格式，如 MP3、WAV 等播放、支持常见的视频格式播放，支持播放列表的创建和保存、支持视频截图，支持常用操作如：播放/暂停、快进/后退、下一首/上一首等。

- 2、实现一个简易的浏览器应用，要求具备浏览器基本功能，访问页面请求、前进、后退、刷新，支持页面打印、可添加书签等。
